# Desktop Timer

OBS配信やデスクトップ作業で使える、Windows向けの小型カウントダウンタイマーです。  
背景透過・常時最前面表示に対応し、設定画面は必要な時だけスライド表示されます。

## 主な機能

- 背景透過のコンパクトなタイマー
- 常時最前面表示
- タイマーの開始・一時停止・リセット
- 時間を `00:00` 形式で設定
- 時分色・パネル色・フレーム色の変更
- タイマー終了時のアラーム音
- MENUボタンと設定画面の左右切り替え
- 設定内容の自動保存
- OBSのウィンドウキャプチャで使用可能

## インストール版の使い方

1. GitHubの **Releases** を開きます。
2. 最新バージョンのインストーラー（`.exe`）をダウンロードします。
3. インストーラーを実行し、画面の案内に従ってインストールします。
4. デスクトップなどに作成された **Desktop Timer** を起動します。

> コード署名を行っていない配布版では、Windowsの警告が表示される場合があります。

## 基本操作

| 操作 | 動作 |
|---|---|
| タイマー表示を左クリック | 開始 / 一時停止 |
| タイマー表示を右クリック | リセット |
| タイマー外周を左ドラッグ | ウィンドウを移動 |
| MENUを左クリック | 設定画面を開く / 閉じる |
| MENUを右クリック | アプリを終了 |
| 設定画面で `Esc` | 設定画面を閉じる |

### キーボード操作

タイマー表示にフォーカスがある状態で使用できます。

| キー | 動作 |
|---|---|
| `Enter` / `Space` | 開始 / 一時停止 |
| `Backspace` / `Delete` | リセット |
| `Esc` | 設定画面を閉じる |

## 設定画面

MENUを左クリックすると、次の設定項目が表示されます。

- **時間**：時と分を入力します
- **時**：タイマー文字の色
- **板**：パネルの色
- **枠**：フレームの色
- **♪**：アラーム音のON / OFF
- **← / →**：MENUと設定画面を左右へ移動

時間・色・アラーム設定・MENU位置は自動保存され、次回起動時にも引き継がれます。

## OBSでの使用方法

1. Desktop Timerを起動します。
2. OBSの「ソース」で `＋` を押します。
3. **ウィンドウキャプチャ**を追加します。
4. キャプチャ対象から **Desktop Timer** を選択します。
5. OBS上で大きさと位置を調整します。

背景は透過されるため、配信画面の上に小型タイマーとして配置できます。

## ソースコードから起動する

ソースコードから使用する場合は、Node.jsとnpmが必要です。

```bash
npm install
npm start
```

## Windows用インストーラーを作成する

```bash
npm run build
```

ビルドが完了すると、`dist` フォルダにインストーラーが作成されます。

## 注意事項

- 現在の配布対象はWindowsです。
- `node_modules` と `dist` はリポジトリへアップロードする必要はありません。
- アプリをビルドする場合は `build/icon.ico` を配置してください。
